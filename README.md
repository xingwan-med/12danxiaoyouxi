# 🎮 12种子弹射击游戏

一个功能完整的HTML5飞行射击游戏，支持无限生命和有限生命模式实时切换。

## 🚀 立即体验

**在线游玩**：https://xingwan-med.github.io/12danxiaoyouxi/

## 🎯 游戏特色

### 🔄 双模式系统
- **♾️ 无限生命模式**：新手友好，专注于射击体验
- **❤️ 有限生命模式**：3条命挑战，考验操作技巧
- **实时切换**：游戏过程中可随时切换模式

## 🎮 操作方法

### 📱 移动设备控制
- **← → 箭头按钮**：控制飞船左右移动
- **🔄 切换子弹按钮**：循环切换12种子弹类型
- **♾️/❤️ 模式切换按钮**：实时切换无限/有限生命模式
- **自动射击**：触摸屏幕或按住移动按钮时自动射击

### 🖱️ 桌面设备控制
- **鼠标点击/拖拽**：控制飞船移动
- **键盘 A/D 或 ←/→**：左右移动飞船
- **键盘 S键 或 空格键**：切换子弹类型
- **键盘 M键**：切换生命模式
- **键盘 P键**：暂停/继续游戏
- **键盘 R键**：重新开始（游戏结束时）
- **双击画面**：全屏/退出全屏
